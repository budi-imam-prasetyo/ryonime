Streaming Anime CLI
Tanpa Iklan!

RyoNime adalah alat CLI untuk streaming anime sub Indo langsung dari terminal. Cepat, ringan, dan tanpa gangguan iklan.

Install Sekarang GitHub

FITUR UNGGULAN

Streaming Cepat

Putar anime langsung dari terminal tanpa buffering. Pengalaman streaming lebih cepat dan ringan.

Auto-Update

Fitur update otomatis tanpa perlu chmod ulang. Dapatkan versi terbaru dengan satu perintah.

Pemutar Multi-Platform

Dukungan untuk MPV, VLC, dan IINA. Otomatis deteksi pemutar yang terinstall di sistem Anda.

Filter Cerdas

Auto-skip video lebih dari 1 jam. Tampilkan durasi sebelum diputar untuk efisiensi waktu.

UI Interaktif

Antarmuka warna-warni dengan navigasi panah. Pengalaman CLI yang lebih menyenangkan dan intuitif.

Tanpa Iklan

Nikmati anime favorit tanpa gangguan iklan. Gratis selamanya dan open source!

INSTALASI CEPAT

Semua Platform

wget https://raw.githubusercontent.com/budi-imam-prasetyo/ryonime/main/ryonime.sh
chmod +x ryonime.sh
./ryonime.sh

Script akan bekerja di Windows (WSL), Linux, dan macOS.

Instalasi Permanen

mkdir -p ~/.local/bin
mv ryonime.sh ~/.local/bin/ryonime
chmod +x ~/.local/bin/ryonime
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
ryonime

Jalankan dari mana saja dengan perintah ryonime

Dependensi Wajib

yt-dlp

pip3 install yt-dlp

Pemutar Video

# MPV (direkomendasikan)
sudo apt install mpv

# VLC
sudo apt install vlc

# IINA (macOS)
brew install iina

Bash

Bash versi 4.0+ (biasanya sudah terinstall di sistem modern)

FITUR AUTO-UPDATE

Cara Update:

1 Melalui Aplikasi

Setelah selesai menonton, pilih opsi [U] Update

2 Via Terminal

ryonime --update

3 Otomatis

Notifikasi akan muncul jika ada versi baru

Keunggulan Update System

  • Backup Otomatis - Script lama disimpan sebagai backup
  • Restore Otomatis - Kembali ke versi lama jika update gagal
  • Pertahankan Permission - Tidak perlu chmod ulang
  • Cek Harian - Otomatis cek versi baru setiap hari
  • One-Click - Proses update dengan satu perintah

PERTANYAAN UMUM

1. Apakah RyoNime legal?

RyoNime adalah alat open source yang mengakses konten publik dari YouTube. Pengguna bertanggung jawab penuh atas penggunaan alat ini sesuai dengan hukum yang berlaku di wilayah mereka.

2. Mengapa saya harus menggunakan CLI?

Antarmuka CLI lebih ringan, cepat, dan hemat sumber daya. Cocok untuk pengguna yang mengutamakan efisiensi dan tidak memerlukan UI grafis.

3. Apakah ada versi Windows?

Ya, RyoNime dapat dijalankan di Windows melalui WSL (Windows Subsystem for Linux). Instalasi WSL diperlukan terlebih dahulu.

4. Bagaimana cara mengganti sumber anime?

Edit file script dan ganti variabel CHANNEL_ID dengan ID channel YouTube lain yang menyediakan konten anime ber-subtitle Indonesia.
